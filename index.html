About This Website

This website is designed to **simulate a blockchain-based voting system**. Users can see information about candidates, vote interactively, and see live vote results in a secure, fun, and engaging frontend experience.

The purpose is to demonstrate a **secure, transparent, and interactive voting system** interface using only frontend technologies. Perfect for learning or demo purposes.
